Flux Releases

Tagged release to be found at Github and on lib.haxe.org .

0.2.5

  • Rework Outcomes, an Outcome is now simply a Future> with extension methods.
  • The Flow module has been deprecated as it was complicated by implementation leakage. Flow is used internally, but the Channel module takes it's places as a simpler public API for pub/sub.
  • Only start_() is required for a Part definition. stop_(mystopfunc) is used to add the stop function.

0.2.4

  • Add Http post/get client methods.
  • Add Http upload handler.
  • Add Bitcoin module.

0.2.3

  • Fix data remoting. Remote clients can now interact with buckets and hashes over haxe remoting.
  • Provide haxe.Http override to standard library which includes a Node.js Http client impl.
